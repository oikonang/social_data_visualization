Assignment 2A: One scatter plot and two datasets

This scatterplot is a representation of the change in number of crimes in SanFrancisco over the years. It seems that if we check the correlation in number of crimes between two different crime categories, and then swipe throught the years we can observe weather the SFPD is doing a good job or not by checking how closer the points in the scatterplot come to the beginning of the axes(0,0). The axes are stable from year to year in order for that transition to be visible. The size of the points relate to the total number of crimes in 2003 and 2015 respectively.

Assignment 2B: Visualizing geodata

This plot represents data related to crimes in San Francisco. Particularly, the coordinates of this mapplot concern only prostitution crimes from 2003 to 2016. Then, unsupervised learning classification is applied using K-nearest neighbor algorithm at the coordinates for 2,3,4,5 and 6 nearest neighbors.

  • On mouse over you may see a preview of the classification along with the centroids per number of neighbors.
  • On mouse click, the preview turns to set mode. That way you may compare classification between 2 different clusterings.
